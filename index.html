Mkdocs环境配置

  • pip install mkdocs

部署方法：“ mkdocs gh-deploy ” 该命令生成静态文件site

并且以默认分支gh-pages（添加参数修改）（--remote-branch main）推送到默认远程仓库origin（添加参数修改）（--remote-name gihub.io）

所以使用指令不加参数时，需要将git pages的Branch选择为gh-pages分支即可显示页面，另外本地的远程仓库名也需要是origin（git remote）
